GIT commands and procedures for using GIT & GitHub

  1. git clone 'GitHub SSH key from GitHub repo page'. This creates a directory that mirrors the GitHub repo
  2. Create your content i.e. HTML, CSS, JS, etc
  3. git add 'filename' or '.' for all files. This adds the file(s) for staging.
  4. git checkout -b feature/'HTML or CSS or JS or whatever'. This creates a new branch (-b) and switches to it.
  5. git push --set-upstream origin feature/html(or whatever). Pushes the changes to the specified branch on GitHub
